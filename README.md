# **Project Title: Hotel Havana Management System**

  **Project Overview:**
  
  Hotel Havana Management System is a graphical project that efficiently manages room bookings, customer records, and payment tracking. It uses an UI with calendar widgets for date selection and supports integrated payment methods.

  **Features:**

  1. Book rooms by entering personal details, and stay dates.
  2. Automates assignment of room number and customer ID.
  3. Admins can search booking records via name or phone.
  4. Payment tracking is integrated within the application.
  5. Whole Graphic UI system is made through Tkinter.

  **Technologies/Tools Used:**

  1. Python 3.13.7
  2. Python Random, Datetime Modules
  3. Python Tkinter Module
  4. Python tkcalender Widget

  **Steps to install and run the project:**
  
  1. Python 3.13.7 should be downloaded.
  2. Tkinter module should be downloaded.
  3. Tkcalender widget should be downloaded.
  4.  1. Run the program.
  2. Open the "Bookings" menu.
  3. Enter your details, check-in date and check-out date.
  4. Submit the booking and note the room number.
  5. Go to "Records" menu to search your booking.
  6. Go to "Payments" menu to track your payments using your phone number.
